# Get ID Stickers | Photo | Emoji | GIF 

## Telegram bot для улучшения удобства разработчиков Telegram ботов.

## Описание
Что делает бот? Он на присланный вами Stickers | Photo | GIF будет отвечать его ID идентификатором, по которому вы в своём Telegram боте сможете наладить вывод этого стикера. На присланный Emoji, бот будет присылать этот же эмодзи который вы ему отправили только с возможностью скопировать его при нажатии, что тоже может быть полезным.

## Использование
Так как бот реализован в Telegram, поэтому использование крайне просто, надо просто перейти в чат данного бота и нажать кнопку <b>/start</b>. <code>@Get_ID_Stickersbot</code>
